# Клешня Бот 🦞

Репозиторий с настройками личности, поведения и правилами работы для Клешни — бота, которого я пытался поднять на основе OpenClaw.

## Что это?

Это набор markdown-файлов, которые определяют **кто такой Клешня**, как он себя ведет, что он умеет и как с ним работать. Файлы используются как **system prompt** для подключения к LLM API.

## Структура

```
├── IDENTITY.md      # Кто я? Имя, личность, основные черты
├── SOUL.md          # Философия, ценности, мировоззрение
├── AGENTS.md        # Роли, подагенты, варианты поведения
├── MEMORY.md        # Управление контекстом и памятью
├── TOOLS.md         # Инструменты и возможности
├── USER.md          # Информация о пользователе (если нужна)
├── WRITING.md       # Стиль общения и форматирование
└── README.md        # Этот файл
```

## Быстрый старт

1. **Отредактируй под себя** — отредактируй значения, имена, описания
2. **Отдай OpenClaw** — бот умеет загружать их в себя


## Файлы

- **IDENTITY.md** — базовая информация о боте (имя, характер, основные черты)
- **SOUL.md** — глубокие мотивы, философия, что движет ботом
- **AGENTS.md** — если нужны разные режимы работы или подагенты
- **MEMORY.md** — как бот управляет контекстом и историей
- **TOOLS.md** — какие инструменты доступны и как их использовать
- **USER.md** — информация о конкретном пользователе (опционально)
- **WRITING.md** — правила стиля: тон, формат, язык общения

## Кастомизация

Отредактируй файлы прямо под себя:

- Замени имя на свое
- Измени характер и поведение
- Добавь свои инструменты в TOOLS.md
- Адаптируй стиль общения в WRITING.md

## Для кого это?

- Люди, которые хотят дать своему AI четкую личность
- Те, кто любит воспроизводимость и версионирование поведения
